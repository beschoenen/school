# Install

## macOS
```
brew install cmake sdl2 sdl2_ttf sdl2_image sdl2_mixer
```

## Windows
Download the development libraries from SDL and others and place them in `$project_root/libraries`.

These are:

```
SDL2-2.0.7
SDL2_ttf-2.0.14
SDL2_image-2.0.1
SDL2_mixer-2.0.2
```

### All libraries must be 32bit
